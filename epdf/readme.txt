Installation: 
- copy directory structure

Library usage:
- GOBO 3.1

Tools usage:
- GOBO gexace for producing multi-platform assembly files
- GOBO geant for automating builds

Variables:
- EPDF			Path of the EPDF library
- SMALLEIFFEL_DIR 	Path of Smalleiffel if you are using it
- GOBO			Path of GOBO

Samples:
- See directory 'examples'
- Look at 'show_epdf' application for a start

Documentation:
- Not for the moment.  See 'doc' directory for miscellaneous files.

Tested:
- ISE 5.0, ISE 5.1, 5.2
- SmallEiffel -0.74, SmartEiffel 1.0
- VisualEiffel 4.1
