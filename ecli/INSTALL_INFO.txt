2. Installation

	2.0 NOTE for Linux/Unix (*n*x) users : 
This package uses unixodbc, avalaible at http://www.unixodbc.org
It has been tested with PostgreSQL 7.0.2.
Under SmallEiffel the program ends with a signal 11 exception (???)
	
	2.1. Building the C library.
A) go to the spec - platform - compiler directory of your choice
B) make the ecli_c.lib (WIN32) or libecli_c.a (*n*x) library 

	2.2. Building the sample
* Linux/Unix) Define the ECLI variable as the ECLI directory
A) SmallEiffel - run the sebuild.bat (WIN32) or sebuild.sh (*n*x) command file
B) ISE - use the Ace.ace file or linux.Ace file
